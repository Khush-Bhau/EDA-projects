# Customer Shopping Behavior — Retail Analytics Portfolio

_End-to-end retail analytics project focused on customer segmentation, promotion effectiveness, and cohort analysis._

## Overview

This project shows how to transform raw customer transactions into business-ready insights using Python, SQL, and Power BI. It includes a reproducible notebook, sample SQL queries, and a Power BI dashboard where available.

## What youll find here

- `Customer_Shopping_Behavior_Analysis.ipynb`: Jupyter notebook with data cleaning, EDA, and visualizations.
- `customer_shopping_behavior.csv`: raw transaction sample used for analysis.
- `customer_behavior_sql_queries.sql`: example SQL queries to compute cohorts and retention metrics.
- `customer_behavior_dashboard.pbix`: Power BI dashboard (if provided).

## How to run

1. Create and activate a Python environment and install dependencies:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install pandas matplotlib seaborn sqlalchemy jupyter nbconvert
```

2. Open and run the notebook interactively or export it to HTML:

```powershell
jupyter nbconvert --to html Customer_Shopping_Behavior_Analysis.ipynb --output CustomerShopping_report.html
```
